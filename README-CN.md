# What Damon Browser - WD浏览器
使用 DYWT 的 EPL 开发的网页浏览器（选择C++的原因是没有EPL选项，并且易语言本身基于C++）
## 优点：
轻量级，支持Flash，简单易用
## 缺点：
不支持英语，不支持HTML5播放器，无法下载文件，不支持多标签页，内核较老（Chrome 65内核），默认UA标识为Windows 10
