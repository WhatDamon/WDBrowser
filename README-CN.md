# What Damon Browser - WD浏览器
用易语言开发的网页浏览器
## 优点：
轻量级，支持Flash(没有广告)，简单易用
## 缺点：
不支持英语，不支持HTML5播放器，无法下载文件，不支持多标签页，内核较老（Chrome 65内核），默认UA标识为Windows 10(无法修改）

## 备注：
